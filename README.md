# Apache Hive (apache-hive)
Apache Hive is a data warehouse software that facilitates reading, writing, and managing large datasets residing in distributed storage using SQL. It provides a SQL-like interface called HiveQL for querying data stored in Hadoop.

## APIs

### Apache Hive
Hive provides HiveServer2 with a Thrift API for JDBC/ODBC connectivity, a Metastore API (Thrift) for metadata management, the Beeline CLI for interactive queries, and WebHCat REST API for job submission and metadata operations.
